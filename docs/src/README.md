# `monkey-c-rs`

<p align="center">
  <img src="./assets/monkey-c-rs-logo.png" alt="monkey-c-rs logo" />
</p>

`monkey-c-rs` is a collection of tools to work with the [Monkey C] programming
language which is [Garmin]'s closed-source programming language used for Garmin
devices. For example projects written in Monkey C, see [awesome-garmin].

## Ecosystem

Although Monkey C has been around for over a decade (founded in 2014), at the
time of writing the tooling we've grown to love for modern programming languages
such as linters and formatters based on static code analysis is almost
completely lacking.

By writing a parser that can understand and represent the Monkey C language I
thought I could write tools that could also help format and analyse the code.
This project is an attempt to close this gap.
